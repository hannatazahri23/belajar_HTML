Atribut Global

"accesskey"

fungsinya untuk menentukan sebuah elemen dapat diakses menggunakan keyboard atau shortcut tertentu


"class" & "id"

fungsinya untuk mengelompokan beberapa elemen pada class atau id tertentu


"dir"

fungsinya untuk menentukan arah tulisannya atau direction


"style"

fungsinya untuk memberikan inline CSS pada sebuah elemen


"tabindex"

fungsinya untuk memberikan urutan pada sebuah form element pada fokus cursor


"title"

fungsinya untuk memberikan judul pada sebuah elemen
